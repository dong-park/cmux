# ~/docs 폴더 조회

`~/docs/` 폴더의 내용을 확인하고 목록을 출력합니다.

## 실행 방법

**단일 Bash 명령어로 실행** (루프/변수 할당 사용 금지):

```bash
# 1. 폴더 목록과 파일 개수를 한 번에 조회
ls -1d ~/docs/*/ 2>/dev/null | xargs -I{} sh -c 'echo "$(basename "{}"):$(find "{}" -type f ! -name ".DS_Store" | wc -l | tr -d " ")"'
```

## 출력 형식

```
📁 ~/docs/
├── 폴더명1/
│   └── N개 파일
├── 폴더명2/
│   └── N개 파일
└── 파일명.md
```

## 인자 처리

- 인자 없음: 전체 폴더 목록 표시
- 인자 있음: 해당 폴더/파일의 상세 내용 (`ls -la` 또는 파일 내용)

---

**조회 대상:** $ARGUMENTS
